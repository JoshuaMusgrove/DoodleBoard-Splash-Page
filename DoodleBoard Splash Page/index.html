New to DoodleBoard?

What is DoodleBoard?

DoodleBoard is a responsive, multi-platform drawing program that runs in a web browser but performs like an App.

How does DoodleBoard work?

The idea is that everyone with a device that can connect to the internet and has touchpad capabilities, can draw on a single board all at one time.

Useable for anything

Designed for use in every situation, the smooth and simple design allows easy use. DoodleBoard can be used from any platform and can be used alone or in a group.

DoodleBoard is Learning

The program is perfect for classroom usage, to help get all the kids to participate.

DoodleBoard is Business

When used in a business setting it can be used to increase the efficiency of meetings exponentially.

DoodleBoard in the Home

DoodleBoard can also be used in your home with simple internet access.
